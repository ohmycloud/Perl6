# 打印指定范围间的行

```perl
use v6;

my $file = open 'flip_flop.txt';
for $file.lines -> $line {
say $line if !( $line ~~ m/^\;/ ff $line ~~ m/^\"/);
}
```
flip_flop.txt 内容如下：
```text
; next is some lines to skip,include this line
fuck fuck fuck
dam dam dam
mie mie mie
" next is subject


There is more than one way to do it
                                -- Larry Wall

We hope Perl6 is wrote by the hole Socfilia
                                -- Larry Wall


; next is some lines to skip,include this line
fuck fuck fuck

                               -- Larry Wall
```
